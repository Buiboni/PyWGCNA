Installation
============

To install PyWGCNA, python version 3.9 or greater is required.


Install from PyPi (recommended)
-------------------------------

Install the most recent release, run

``pip install PyWGCNA``


Install with the most recent commits
------------------------------------

git cloning the `PyWGCNA repository <https://github.com/mortazavilab/PyWGCNA>`_, going to the PyWGCNA directory, run

``pip install .``
